# Information <a name="information"></a>  
Welcome to the Rock, Paper, Sissors game. With this game you don't need to have a friend around to play.

## Installation <a name="install"></a>  

The game can be downloaded as a ZIP using <a href="https://github.com/JDBrendel159/Rock-Paper-Scissors/archive/release.zip">this link.</a>

Extract the ZIP to a folder of your choice, and open "run.bat" to begin the game.  
<br>   

## Game Rules <a name="rules"></a>  

<img src="https://en.wikipedia.org/wiki/Rock_paper_scissors#/media/File:Rock-paper-scissors.svg" alt="Basic game rules explanation">


Rock beats scissors, scissors beats paper, paper beats rock, and ties are excluded. 

There must be an odd number of rounds.

The game is complete when a player wins a majority of the games compared to the total number of rounds. 
<br>  

## Tips <a name="tips"></a>  
* To exit the game press CTRL-C and then Y.
* The game will restart at the end of each game.
* Another version of the readme file comes with the download.  
<br>  

## Troubleshooting <a name="problems"></a>  
The Zip file can be extracted with either of these <a href="https://www.rarlab.com/download.htm">WinRAR</a> or <a href="https://www.7-zip.org">7Zip.</a>

If the run file does not work Java may need to be updated.  THis link will lead you to an <a href="https://www.java.com/en/download">update.</a>
